Hexo+Github 搭建属于自己的博客系列教程三：Hexo常用命令与Markdown常用语法

1，Hexo 常用的命令

1.1 常用指令与简写


hexo new "标题名" #新建文章      可简写为：hexo n "标题名"

hexo new page "页面名" #新建页面

hexo generate #生成静态页面至public目录      可简写为： hexo g

hexo server #开启预览访问端口      可简写为： hexo s

hexo deploy #部署至服务器      可简写为： hexo d

hexo clean #清除缓存，会删除以前生成的public目录中内容

hexo qiniu s #扩展 ，安装七牛hexo-qiniu-sync 插件后可使用，用来上传图片

1.2 复合指令使用

1.2.1 跟换端口预览

hexo server -p 5000 跟换端口预览（在默认端口被占用的使用可使用）

可简写为： hexo s -p 5000

1.2.2 生成后本地预览

hexo s -g 生成后本地预览

1.2.3 生成后直接发布

hexo d -g 生成后直接发布

2，Markdown常用的语法

2.1 标题 ‘#’

# 1级标题 最大

# 2级标题

# 3级标题

# 4级标题

# 5级标题

# 6级标题

2.2 粗体，斜体 ‘*’ or ‘_’

前后一个 * 或者 _ 为斜体 示例 *斜体* _斜体_

前后两个 * 或者 _ 为粗体 示例 **粗体** __粗体__

2.3 链接

格式一： [链接名称](网址)

格式二： 链接名称<网址>

2.4 图片名称

格式：![图片名称](图片地址)

2.5 无序列表 ‘-’ or ‘+’or‘*’

格式一：- 文本 减号+空格 文本

格式二：+ 文本 加好+空格 文本

格式三： * 文本 星号+空格 文本

2.6 有序列表 ‘数字.’

格式： 数字. 文本 数字+英文句点+空格 文本（数字不影响实际的效果）

2.7 引用 ‘>’

格式： > 文本

2.8 分隔符 ‘—’

格式：---

2.9 区块（4个空格）

格式：缩进4个空格 文本
